# 🛍️ Bozorgo

Bozorgo is a modern full-stack e-commerce platform built to provide a fast, responsive, and user-friendly shopping experience. The project follows a scalable feature-based architecture and demonstrates modern web development practices on both the frontend and backend.

## ✨ Features

- 🔐 User Authentication & Authorization
- 👤 User Profile Management
- 🛒 Shopping Cart
- ❤️ Wishlist
- 📦 Product Catalog
- 🔍 Search & Filtering
- 📄 Product Details
- 🗂️ Category Management
- 💳 Checkout Process
- 📜 Order History
- 📱 Responsive Design
- ⚡ Fast Data Fetching & Caching
- 🔄 State Management with Redux Toolkit

## 🛠️ Tech Stack

### Frontend

- React
- JavaScript (ES6+)
- Redux Toolkit
- TanStack Query
- React Router
- Axios
- Tailwind CSS
- Material UI

### Backend

- Python
- Django
- Django REST Framework

### Database

- PostgreSQL

### Development Tools

- Git & GitHub
- Vite
- Postman

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/bozorgo.git
```

### Install dependencies

```bash
cd client
npm install
```

### Start the development server

```bash
npm run dev
```
